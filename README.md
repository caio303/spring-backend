# Authentication and Authorization CRUD

This is basically a REST API that registers, signs in, and authenticates users.

For building it I have used these languages and technologies:

* Java 17
* Spring Boot
* Hibernate
* JPA
* JWT
* Swagger Documentation
* Insomnia
